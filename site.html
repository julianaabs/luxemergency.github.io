Luz de Emergência


    Projeto da disciplina Sistemas Operacionais com a BeagleBone Black



    A plataforma


    A BeagleBone Black é um dispositivo de desenvolvimento de baixo custo que utiliza o Linux como sistema operacional. Essa plataforma está sendo utilizada nas aulas práticas da disciplina de Sistemas Operacionais do Bacharelado em Tecnologia da Informação da Universidade Federal do Rio Grande do Norte, lecionada pelo professor Ivanovitch Silva. O código do projeto se encontra hospedado no Github.

    O projeto

    Descrição do projeto

    O projeto desenvolvido visa simular uma luz de emergência, que é acionada quando não se detecta iluminação no ambiente onde se encontra a BeagleBone. O código do sistema foi feito utilizando a linguagem Python e foram utilizados os seguintes acessórios na plataforma:
    Led vermelho
    Led RGB
    Potenciômetro
    Fotoresistor
    Botão

    O funcionamento do sistema começa quando o botão é pressionado. Logo quando iniciado, o fotoresistor é responsável por detectar se há ou não iluminação no ambiente. Se houver, o LED vermelho irá acender indicando que há luz no local e que o sistema está sendo executado. Assim que o fotoresistor registrar a ausência de iluminação, o LED RGB será acionado e a cor dele irá variar de acordo com o valor registrado no potenciômetro:
    De 0 a 1365: Azul
    De 1366 a 2730: Verde
    De 2731 a 4095: Vermelho
